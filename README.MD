# Proyecto Guía Final

Proyecto base para el trabajo semestral de la asignatura **Desarrollo FullStack 1** en **DUOC UC - Sede San Joaquín**.  
Este repositorio busca entregar una guía práctica y modular para el desarrollo de aplicaciones **FullStack** con tecnologías modernas.

---

## Tecnologías principales

- **Spring Boot 4**
- **Java 21**
- **MySQL**
- **Docker**

---

## Subproyectos incluidos

1. **Login**  
   Módulo de autenticación con manejo de **Auth** y **JWT** para la gestión de usuarios y seguridad.

2. **DemoCars**  
   Módulo encargado de listar y administrar los vehículos disponibles en el sistema.

3. **DemoBranches**  
   Módulo que gestiona las sucursales disponibles, incluyendo información de ubicación y disponibilidad.

## Starters utilizados

- **spring-boot-starter-web**  
  Para la construcción de APIs REST y manejo de controladores.

- **spring-boot-starter-data-jpa**  
  Para la integración con JPA y persistencia en MySQL.

- **spring-boot-starter-security**  
  Para la implementación de seguridad y autenticación con JWT.

- **spring-boot-starter-test**  
  Para pruebas unitarias e integración.

---

## Notas finales

Este proyecto está diseñado como **guía práctica** para estudiantes, con el objetivo de reforzar conceptos del patrón **CSR**, modularidad y despliegue en entornos **Dockerizados**.  
Cada subproyecto puede ser extendido y adaptado según las necesidades del trabajo semestral.
